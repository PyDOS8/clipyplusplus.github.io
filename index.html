Cli++Py

CLI++Py is an open-source terminal written in C++ and Python

CLI++Py can be used for:

  • Making folders
  • Making files
  • deleting files
  • deleting folders
  • listing files
  • listing folders
  • pinging
  • checking if a file or folder exists
  • running programs
  • Rename files
  • IP Configuration
  • Display information about active network connections
